Computer Science 3
Client-Side Web Development

The course is designed to employ output-based learning and focuses on client-side web development using HTML, CSS and JavaScript. The client-side programming exposes the students in writing code that will run on the client, and is done in languages that can be executed by the browser, such as JavaScript as it motivates the students to collaborate and participate in various social issues, and learn the value of teamwork to accomplish various projects.
